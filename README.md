## Модель
Математичне моделювання та оптимізація маршрутів доставки для мінімізації витрат палива (VRP)

## Автор

## Опис моделі
Дана модель призначена для мінімізації загальних витрат на логістику шляхом розрахунку оптимальних маршрутів. 
Цільова функція:
$$Z = \sum_{i,j,k} (D_{ij} \cdot C_k \cdot x_{ijk}) \to \min$$

Основні обмеження:
* Кожен клієнт відвідується рівно один раз.
* Сумарний попит на маршруті не перевищує вантажопідйомність авто $Q_k$.
* Кожне авто починає та закінчує шлях у депо (вузол 0).
